# Testing task "Pineapple" for Magebit

## How to start a project locally:
Download or clone:
```sh
git@github.com:vadigru/pineapple.git
```
Install project on local computer (node.js required):
```sh
npm install
```
Run project:
```sh
npm run start
```
Create production build:
```sh
npm run build
```
